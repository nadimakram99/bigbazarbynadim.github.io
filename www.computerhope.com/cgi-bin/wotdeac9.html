Screenshot

Updated: 09/03/2019 by Computer Hope

Alternatively referred to as a print screen, screen dump, snapshot, or screen capture, a screen shot or screenshot is an image of what's currently displayed on a screen. Screenshots can be created by a software program or operating system. Once a screenshot is captured, it can be saved as an image file, such as GIF or JPEG where it can be opened to manipulate, print, or saved.
